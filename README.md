# ğŸ“– SmartSpender <a name="about-project"></a>

**SmartSpender** is a mobile web application where you can manage your budget: you have a list of purchases associated with a category, so that you can see how much money you spent and on what.

## ğŸ—ï¸ Key Features <a name="key-features"></a>

- Allows users to register and log in
- Displayed data is private to the user
- Add and see your categories
- Add and see your purchases associated with a category

## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- [Install Ruby](https://www.ruby-lang.org/en/documentation/installation/)
- Install the [rails gem](https://rubygems.org/gems/rails)

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:dicodiaz/budget-app.git
  cd budget-app
```

### Usage

To run the project, execute the following commands:

```sh
  rails server
```
